## 目标
实现无类型语言的灵活性并保持类型安全

![alt text](./etc/abstract-document.png "Abstract Document Traits and Domain")


## 适用性
使用抽象文档模式场景:
* 需要动态添加新的属性  
* you want a flexible way to organize domain in tree like structure
* 你想要更松散耦合的系统
